# صور اللوجوهات البنكية

## كيفية استخدام الصور

1. **ضع الصور في مجلد `public/`**:
   - `logo.png` - صورة عامة للموقع أو التطبيق
   - `misr.png` - لوجو بنك مصر
   - `elahly.png` - لوجو البنك الأهلي المصري

2. **الصيغ المدعومة**:
   - PNG (موصى به)
   - JPG/JPEG
   - SVG (للرسومات المتجهة)
   - WebP (للويب الحديث)

3. **أحجام الصور الموصى بها**:
   - **عرض**: 100-200 بكسل
   - **ارتفاع**: 100-200 بكسل
   - **دقة**: 72-150 DPI

## ملاحظات

- الكود معد للتعامل مع الصور المفقودة تلقائياً
- إذا لم يتم العثور على الصورة، سيعود إلى الإيموجي الافتراضي
- تأكد من أن أسماء الملفات تطابق تماماً:
  - `logo.png` (حساسة لحالة الأحرف)
  - `misr.png`
  - `elahly.png`

## مثال على الهيكل

```
public/
├── logo.png
├── misr.png
├── elahly.png
└── ...
```

## تحديث الكود

تم تحديث المكونات التالية لدعم الصور:

1. **BanksDisplay Component** - عرض الصور في بطاقات البنوك
2. **CalculatorClient Component** - عرض الصور في قائمة الاختيار
3. **Bank Data JSON** - مسارات الصور في بيانات البنوك
